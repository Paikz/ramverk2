# ramverk2

## Installation

Clone the repository  
`npm install`  
`npm start`  
`make test`  

## Docker

Start up an express server with Docker:  
`npm run node[version]`  
    Currently supported: node 6,7,8,9

Stop docker containers:  
`npm run stop` || `npm stop`
